**JAMZ.TECH Audio Application **

Our collaborators: MARY SANNAPU, TANISHI KUMAR, ANOUSHKA BARUA

We created an audio application using C++ and the JUCE audio framework. 
Our application has a user-safe login/logout system. The stakeholders may record, edit, play, set a price to their audio, and even visualize their 
audio in a clustermap feature. We have implemented an RBAC system where users have access to certain facilities based on the type of the user account
(owner or guest account). 

There are additional interesting features to explore and learn more about. Please feel free to read our report and view our code. :)
